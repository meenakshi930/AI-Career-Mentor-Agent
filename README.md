# 🚀 AI Career Mentor Agent

A Streamlit-based AI application that analyzes a user's current skills, identifies skill gaps, and provides personalized career guidance using Google Gemini AI.

## ✨ Features

* Skill Gap Analysis
* Personalized Learning Roadmap
* AI-Powered Career Mentor Advice
* Project Recommendations
* Certification Suggestions
* Career Readiness Assessment

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* Prompt Engineering

## ⚙️ Setup

### 1. Clone the Repository

```bash
git clone https://github.com/meenakshi930/AI-Career-Mentor-Agent.git
cd AI-Career-Mentor-Agent
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Create a `.env` File

Create a `.env` file in the project root directory and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the Application

```bash
streamlit run app.py
```

Open your browser and visit:

```text
http://localhost:8501
```
